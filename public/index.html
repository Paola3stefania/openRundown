UNMute API

This is a serverless API project. All endpoints are under /api/.

Available Endpoints

GET/POST /api/cron/daily-sync

Automated daily sync (requires CRON_SECRET authentication)

POST /api/tools/tool

Generic tool execution endpoint

POST /api/tools/sync

Manual sync endpoint

POST /api/tools/export

Manual export endpoint

GET /api/tools/status

Status and statistics endpoint

All endpoints require authentication via Authorization: Bearer <API_KEY> header.
